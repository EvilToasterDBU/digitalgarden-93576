## Первый запуск
Когда запустится майн жмем вторую кнопку ![](/img/user/img/fs/01.png)
При подключении нас попросит связать дискорд-аккаунт с майнкрафт-аккаутном![](/img/user/img/fs/02.png)
Для этого пишем код боту в лс в дискорде для привязки ![](/img/user/img/fs/03.png)
Подключаемся еще раз![](/img/user/img/fs/04.png)
Соглашаемся на загрузку![](/img/user/img/fs/05.png)
Ждем окончания загрузки![](/img/user/img/fs/06.png)
Теперь нужно перезапустить майн, закрываем сборку нажав **Да, закрыть игру**![](/img/user/img/fs/07.png)
Запускаем сборку снова из вашего лаунчера и получаем готовую к игре сборку ![](/img/user/img/fs/08.png)
При первом подключении вас попросят зарегаться![](/img/user/img/reg.png)
Регистрируемся при помощи команды `/reg пароль пароль`
При повторном входе вас попросят залогиниться с вводом `/login пароль`
**Если вы лицушный игрок**, после этого 2 раза выполните команду `/premium`![](/img/user/img/prem01.png)
После этого вас выбросит из игры и при следующем входе вы будете автоматически авторизованы без ввода пароля
![](/img/user/img/prem02.png)
Если будете играть с дальней прорисовкой (мод voxy уже предустановлен) переходите в раздел [[firstrun#Настройка Voxy\|#Настройка Voxy]]
